# Portfolio Summary

FloodSense Lokoja is a Python-based flood risk intelligence and decision-support system for Lokoja, Kogi State, Nigeria.

The project combines GIS terrain analysis, hydrological modelling, flood susceptibility mapping, Sentinel-1 SAR validation, exposure analysis, and priority ranking for flood intervention planning.

Portfolio strengths:

- Reproducible Python geospatial workflow.
- Clear modular package design.
- Transparent flood susceptibility model.
- Historical satellite validation using Sentinel-1 SAR.
- Exposure quantification for buildings, roads, and population.
- Dashboard-ready outputs for WebGIS communication.

The project starts with Lokoja only and can later be adapted to other flood-prone Nigerian cities.
