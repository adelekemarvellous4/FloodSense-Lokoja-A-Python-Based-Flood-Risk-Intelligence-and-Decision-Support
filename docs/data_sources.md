# Data Sources

Place raw datasets in these folders:

- Boundary: `data/raw/boundary/`
- DEM: `data/raw/dem/`
- Rainfall: `data/raw/rainfall/`
- Sentinel-1 pre/post flood rasters: `data/raw/sentinel1/`
- Land cover: `data/raw/landcover/`
- Buildings: `data/raw/buildings/`
- Roads: `data/raw/roads/`
- Population raster: `data/raw/population/`

Recommended sources include administrative boundary data, Copernicus DEM or SRTM, CHIRPS rainfall, Sentinel-1 SAR, ESA WorldCover, OpenStreetMap buildings/roads, and WorldPop population rasters.

Record source URL, acquisition date, spatial resolution, CRS, license, and preprocessing notes before analysis.
